# Steam Hour Booster Bot

Telegram бот для накрутки часов в играх Steam с поддержкой управления тремя аккаунтами и Steam Guard.

## 🚀 Возможности

- ⚡ Управление через inline кнопки
- 👥 Поддержка до 3 аккаунтов одновременно
- 🎮 Автоматическая накрутка часов для выбранных игр
- 🔐 **Полная поддержка Steam Guard** (мобильный аутентификатор и email)
- 💾 Сохранение сессий для повторного входа без 2FA
- 📊 Статистика работы аккаунтов
- 🔒 Безопасное хранение токена в .env

## 📋 Требования

- Python 3.8+
- Steam аккаунты (рекомендуется использовать отдельные аккаунты для накрутки)
- Telegram бот (получить у @BotFather)
